# Working with Agents

When working with agents on this project, persist any plans under `.claude/plans/` using the following format:

```
.claude/plans/YYYY-MM-DD-<description>.md
```

## Examples

- `.claude/plans/2026-03-17-prototype-refactor.md`
- `.claude/plans/2026-03-15-add-api-endpoints.md`
- `.claude/plans/2026-03-10-performance-optimization.md`

This naming convention makes it easy to find plans by date and quickly understand what work was planned.
